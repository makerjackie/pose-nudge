## ✨ Temel Özellikler

Pose Nudge, web kameranızı kullanarak duruşunuzu gerçek zamanlı olarak analiz eden ve kaplumbağa boynu gibi duruş sorunları tespit edildiğinde bildirim göndererek doğru duruşunuzu korumanıza yardımcı olan güçlü bir masaüstü uygulamasıdır.

*   **📹 Gerçek Zamanlı Duruş Analizi**: Web kamerası tabanlı gerçek zamanlı duruş izleme ve yapay zeka destekli analiz
*   **🦴 Kaplumbağa Boynu Tespiti**: Boyun ve omuz çizgisi açılarını hesaplayarak kaplumbağa boynu durumunu tespit eder
*   **🔔 Akıllı Bildirimler**: Duruş sorunları tespit edildiğinde tarayıcı bildirimleri ve iyileştirme önerileri
*   **📊 Duruş Puanı**: Mevcut duruş durumunuzu 0-100 puan aralığında gösterir
*   **📈 İstatistik Paneli**: Duruş iyileştirme ilerlemenizi ve oturum kayıtlarınızı görüntüleyin
*   **⚙️ Kişiselleştirilmiş Ayarlar**: Bildirim aralıkları, hassasiyet ve analiz sıklığı gibi özelleştirilebilir ayarlar

## 👨‍💻 Geliştiriciler İçin

Katkıda bulunmakla ilgileniyorsanız, projeyi yerel olarak kurmak için bu rehberi takip edin.

### Ön Gereksinimler

- [Node.js](https://nodejs.org/) (v18 veya üzeri)
- [Rust](https://www.rust-lang.org/) (v1.70.0 veya üzeri)
- [Git](https://git-scm.com/)

### Kurulum ve Çalıştırma

```bash
# 1. Projeyi klonlayın
git clone https://github.com/dduldduck/pose-nudge.git
cd pose-nudge

# 2. Node.js bağımlılıklarını yükleyin
npm install

# 3. Geliştirme modunda çalıştırın
npm run tauri dev
```

### Proje Yapısı
```
pose-nudge/
├── src/                    # React Ön Yüz
│   ├── components/         # UI Bileşenleri
│   │   ├── ui/            # shadcn/ui Bileşenleri
│   │   ├── Dashboard.tsx   # Gösterge Paneli
│   │   ├── WebcamCapture.tsx # Web Kamerası Bileşeni
│   │   └── SettingsPage.tsx # Ayarlar Sayfası
│   ├── lib/               # Yardımcı Fonksiyonlar
│   ├── locales/           # Çoklu Dil Desteği
│   └── App.tsx            # Ana Uygulama Bileşeni
├── src-tauri/             # Rust Arka Uç
│   ├── src/
│   │   ├── main.rs        # Ana Arka Uç Mantığı
│   │   ├── pose_analysis.rs # Duruş Analiz Motoru
│   │   └── notifications.rs # Bildirim Sistemi
│   ├── Cargo.toml         # Rust Bağımlılıkları
│   └── tauri.conf.json    # Tauri Yapılandırması
├── models/                # Yapay Zeka Model Dosyaları
├── public/                # Statik Dosyalar
└── locales/               # Yerelleştirme Dosyaları
```

## 🛠️ Teknoloji Yığını

-   **Framework**: Tauri (Rust + React)
-   **Ön Yüz**: React 19, TypeScript, Tailwind CSS 4
-   **Arka Uç**: Rust, Tauri 2
-   **Yapay Zeka/Makine Öğrenimi**: YOLO-Pose Modeli (gelecekte entegre edilecek)
-   **Derleme/Dağıtım**: Tauri CLI
