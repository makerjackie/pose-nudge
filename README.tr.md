# OnePosture

> OnePosture, özgün [Pose Nudge](https://github.com/DDULDDUCK/pose-nudge) projesinden geliştirilen, gizlilik odaklı bir duruş uygulamasıdır. Özgün katkı sahipleri ve AGPL-3.0 geçmişi korunur.

<p align="center">
  <img src="public/logo.png" alt="OnePosture logosu" width="150">
</p>

Kamera görüntüleri yalnızca cihaz üzerinde analiz edilir. Temel izleme, sistem bildirimleri ve üstte yüzen hatırlatıcı ücretsizdir. OnePosture Pro, ekran karartma hatırlatıcısını açan tek seferlik bir satın alımdır.

## Temel özellikler

- Çoklu kare ve kişisel referans duruşla yerel analiz
- Sistem bildirimi, yüzen pencere, ses ve ekran karartma kanalları
- Menü çubuğunda görünür izleme durumu
- İngilizce, Korece, Japonca, Basitleştirilmiş/Geleneksel Çince ve Türkçe

## İndirme ve geliştirme

- [En son macOS sürümü](https://github.com/makerjackie/pose-nudge/releases/latest)
- Geliştirme: `npm install && npm run tauri dev`
- Doğrulama: `npm run check:localization && npm run build`
